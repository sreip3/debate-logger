# About the Interface

**Setup and Settings**
* **Setup Menu**: Allows selection of specific folders for **Notes**, **Speech**, and **Feedback**.
* **Settings Configuration**:
    * **Redo Setup**: Option to re-initialize system configurations.
    * **Appearance**: **Dark/Light mode** toggling.
    * **Font**: Integrated **font change** settings.

**Core Functions**
* **Speech Recording**: Native audio recording with automated upload to the designated **Drive folder**.
* **Feedback System**:
    * Records and uploads feedback to **Drive**.
    * **Transcription**: Automatically transcribes feedback to a **Google Sheet**.
    * **Image Management**: Photo system creates **sub-folders** and uploads pictures into it.
* **Notes Module**:
    * Photo system creates **sub-folders** and uploads pictures into it.
* **Metadata input for**: *Date*, *Motion*, *Style*, *Position*, and *Score*.

**Utilities**
* **Timer**:
    * Functional controls: *Start*, *Reset*, *Pause*, and *Resume*.
    * Adjustment: Capability to **add** or **subtract** time.
* **Management**:
    * **Auto-save** functionality for data saving.
    * **Delete function** to clear data.
* **Change/edit**: **Edit by code** feature for manual configuration.
